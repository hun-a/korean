# 022일차 - 복합모음 + 받침 통합

## 오늘 목표
- 복합모음과 받침이 함께 있는 글자를 읽는다.
- 어려운 단어를 따라 써 본다.

## 핵심 학습
- 단어: 괜찮아, 외출, 귀염, 꿈, 뿔

## 15분 수업 구성
### 1단계 (5분) 자음/모음 읽기
- `ㅚ`, `ㅘ`, `ㅝ`와 받침 글자 조합 읽기
- 어려운 글자는 음절로 끊어 읽기

### 2단계 (5분) 단어 읽기
- 단어 5개를 음절 단위로 읽기
- 느린/보통 속도 버튼으로 반복 읽기

### 3단계 (5분) 따라 쓰기
- 단어 2개 선택 따라 쓰기
- 틀린 글자만 다시 쓰는 교정 모드 수행

## 웹 서비스 적용 가이드
- 좌측 패널: 음절 분해 보기(`괜-찮-아`)
- 우측 캔버스: 어려운 글자 확대 트레이싱
- 성공 피드백: 교정 완료 시 폭죽 + 칭찬 배너

## 확인 체크
- 어려운 단어도 음절로 끊어 읽는다.
- 틀린 글자를 스스로 고쳐 쓴다.
