# 020일차 - 쌍자음 `ㄲ`, `ㄸ`, `ㅃ`, `ㅆ`, `ㅉ`

## 오늘 목표
- 쌍자음을 읽고 기본 소리와 비교한다.
- `까`, `따`, `빠`, `싸`, `짜`를 읽고 쓴다.

## 핵심 학습
- 자모: `ㄲ`, `ㄸ`, `ㅃ`, `ㅆ`, `ㅉ`
- 단어: 딸기, 빵, 싸움, 짝

## 15분 수업 구성
### 1단계 (5분) 자음/모음 읽기
- `ㄱ-ㄲ`, `ㄷ-ㄸ`처럼 짝 비교 읽기
- 쌍자음은 힘주어 읽는 연습

### 2단계 (5분) 단어 읽기
- `딸기`, `빵`, `싸움`, `짝` 읽기
- 쌍자음 글자 찾기 게임

### 3단계 (5분) 따라 쓰기
- 쌍자음 5개 + 음절 3개 따라 쓰기
- 쌍자음 획 누락 없이 쓰면 폭죽 효과

## 웹 서비스 적용 가이드
- 좌측 패널: 기본자음/쌍자음 비교 슬라이드
- 우측 캔버스: 두 획 동시 강조 가이드
- 성공 피드백: 쌍자음 전부 성공 시 특별 배지

## 확인 체크
- 쌍자음을 구별해 읽는다.
- 쌍자음 획을 빠뜨리지 않고 쓴다.
