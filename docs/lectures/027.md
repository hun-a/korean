# 027일차 - 짧은 이야기 읽기 2

## 오늘 목표
- 짧은 이야기를 끊어 읽으며 의미를 이해한다.
- 기억에 남는 문장을 따라 쓴다.

## 핵심 학습
- 이야기: `토끼가 당근을 먹어요. 배가 불러요. 낮잠을 자요.`
- 핵심 단어: 토끼, 당근, 낮잠

## 15분 수업 구성
### 1단계 (5분) 자음/모음 읽기
- `토`, `끼`, `당`, `근`, `잠` 읽기
- 쌍자음/받침 있는 음절 집중

### 2단계 (5분) 단어 읽기
- 단어 3개 읽고 문장 3개 읽기
- 읽은 문장과 그림 연결 퀴즈

### 3단계 (5분) 따라 쓰기
- 좋아하는 문장 1개 선택 따라 쓰기
- 단어 1개 추가로 교정 쓰기

## 웹 서비스 적용 가이드
- 좌측 패널: 문장별 오디오 재생 버튼
- 우측 캔버스: 선택 문장 트레이싱
- 성공 피드백: 선택 문장 완성 시 폭죽

## 확인 체크
- 문장 의미를 그림으로 고른다.
- 문장 1개를 정확히 쓴다.
