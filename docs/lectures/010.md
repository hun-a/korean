# 010일차 - 자음 `ㅈ`, `ㅊ`, `ㅋ`, `ㅌ`, `ㅍ`, `ㅎ`

## 오늘 목표
- 남은 기본 자음을 읽고 쓸 수 있다.
- `자/차/카/타/파/하`를 읽는다.

## 핵심 학습
- 자모: `ㅈ`, `ㅊ`, `ㅋ`, `ㅌ`, `ㅍ`, `ㅎ`
- 단어: 자동차, 코끼리, 토끼, 포도, 하늘

## 15분 수업 구성
### 1단계 (5분) 자음/모음 읽기
- 자음 6개 읽고 `ㅏ` 결합 음절 읽기
- 거센소리(`ㅊㅋㅌㅍ`)를 크게 읽기

### 2단계 (5분) 단어 읽기
- `자동차`, `코끼리`, `토끼`, `포도`, `하늘` 읽기
- 카드 랜덤 등장 시 빠르게 읽기

### 3단계 (5분) 따라 쓰기
- 자음 6개 중 4개 선택 따라 쓰기 + 음절 4개 쓰기
- 완료 시 폭죽 1회 + 칭찬 음성

## 웹 서비스 적용 가이드
- 좌측 패널: 거센소리 아이콘(바람 효과)
- 우측 캔버스: 쓰기 완료도 진행바
- 성공 피드백: 진행바 100% 시 축하 화면

## 확인 체크
- 기본 자음을 대부분 읽는다.
- 선택 음절을 획순대로 쓴다.
