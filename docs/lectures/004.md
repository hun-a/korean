# 004일차 - 확장 모음 `ㅑ`, `ㅕ`, `ㅛ`, `ㅠ`

## 오늘 목표
- `ㅏ/ㅓ/ㅗ/ㅜ`와 짝 모음(`ㅑ/ㅕ/ㅛ/ㅠ`)을 연결한다.
- `야`, `여`, `요`, `유`를 읽고 쓴다.

## 핵심 학습
- 자모: `ㅑ`, `ㅕ`, `ㅛ`, `ㅠ`
- 단어: 야구, 여우, 요트, 유리

## 15분 수업 구성
### 1단계 (5분) 자음/모음 읽기
- 짝 읽기: `ㅏ-ㅑ`, `ㅓ-ㅕ`, `ㅗ-ㅛ`, `ㅜ-ㅠ`
- 점이 하나 더 생긴 모양 강조

### 2단계 (5분) 단어 읽기
- `야구`, `여우`, `요트`, `유리`를 큰 소리로 읽기
- 랜덤 카드 뒤집기 퀴즈 4문제

### 3단계 (5분) 따라 쓰기
- `ㅑ ㅕ ㅛ ㅠ`와 `야 여 요 유` 따라 쓰기
- 8개 중 6개 이상 맞으면 폭죽 + 칭찬 문구

## 웹 서비스 적용 가이드
- 좌측 패널: 짝 모음 비교 카드
- 우측 캔버스: 점 위치 오답 시 즉시 하이라이트
- 성공 피드백: 오답 1회 후 재도전 시 추가 칭찬

## 확인 체크
- 짝 모음을 보고 소리를 낸다.
- `야/여/요/유`를 2회 이상 바르게 쓴다.
