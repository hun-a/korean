# 011일차 - 모아쓰기 1 (`가~하` 기본 음절)

## 오늘 목표
- 초성+중성 구조를 이해한다.
- 자음과 `ㅏ`를 결합한 기본 음절을 읽고 쓴다.

## 핵심 학습
- 자모: 기본 자음 + `ㅏ`
- 단어: 가구, 나무, 다리, 바지

## 15분 수업 구성
### 1단계 (5분) 자음/모음 읽기
- 자음 카드와 `ㅏ`를 붙여 `가 나 다 ... 하` 읽기
- 글자 블록 조합 애니메이션 보기

### 2단계 (5분) 단어 읽기
- 받침 없는 2글자 단어 4개 읽기
- 단어를 두 글자로 끊어 읽기 연습

### 3단계 (5분) 따라 쓰기
- `가 나 다 라 마 바 사 아` 중 6개 따라 쓰기
- 단어 2개 따라 쓰기

## 웹 서비스 적용 가이드
- 좌측 패널: 조합 전/후(자모 -> 음절) 비교
- 우측 캔버스: 음절 칸 자동 정렬
- 성공 피드백: 단어 완성 시 박수 + 스티커

## 확인 체크
- 자모를 보고 한 글자로 모아 읽는다.
- 기본 음절을 6개 이상 바르게 쓴다.
