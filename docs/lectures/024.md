# 024일차 - 질문과 대답 문장

## 오늘 목표
- 짧은 질문/대답 문장을 읽는다.
- `~요` 문장 끝맺음을 익힌다.

## 핵심 학습
- 문장: `이건 뭐예요?`, `사과예요.`, `누가 와요?`, `엄마가 와요.`

## 15분 수업 구성
### 1단계 (5분) 자음/모음 읽기
- 문장 핵심 음절(`뭐`, `예`, `와`) 읽기
- 물음표 문장 억양 따라 읽기

### 2단계 (5분) 단어 읽기
- 질문/대답 문장에서 단어 먼저 읽기
- 문장 짝 맞추기 놀이(질문-대답 연결)

### 3단계 (5분) 따라 쓰기
- 질문 1개 + 대답 1개 따라 쓰기
- `?`와 `.`까지 정확히 쓰기

## 웹 서비스 적용 가이드
- 좌측 패널: 질문-대답 카드 매칭 UI
- 우측 캔버스: 문장 부호 힌트 제공
- 성공 피드백: 매칭 성공 시 박수 애니메이션

## 확인 체크
- 질문과 대답 문장을 구분해 읽는다.
- 문장 부호를 포함해 쓴다.
