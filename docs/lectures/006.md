# 006일차 - 복합모음 `ㅘ`, `ㅝ`

## 오늘 목표
- `ㅘ`, `ㅝ`를 소리로 구분한다.
- `와`, `워`가 들어간 단어를 읽고 쓴다.

## 핵심 학습
- 자모: `ㅘ`, `ㅝ`
- 단어: 와이, 워터, 과자, 원숭이

## 15분 수업 구성
### 1단계 (5분) 자음/모음 읽기
- `ㅗ+ㅏ=ㅘ`, `ㅜ+ㅓ=ㅝ` 원리 소개 후 읽기
- `와/워` 반복 발음 연습

### 2단계 (5분) 단어 읽기
- `과자`, `원숭이`, `와이`, `워터` 읽기
- 듣고 맞는 단어 고르는 클릭 게임

### 3단계 (5분) 따라 쓰기
- `ㅘ`, `ㅝ`, `와`, `워`, `과`, `원` 따라 쓰기
- 성공 시 박수 + 캐릭터 점프 애니메이션

## 웹 서비스 적용 가이드
- 좌측 패널: 결합식(`ㅗ+ㅏ`) 시각화
- 우측 캔버스: 두 모음 결합 포인트 점선 제공
- 성공 피드백: 단어 완성마다 스탬프 지급

## 확인 체크
- `와/워` 소리를 구별한다.
- 복합모음을 포함한 글자를 획순대로 쓴다.
