# 013일차 - 받침 없는 단어 읽기 1

## 오늘 목표
- 받침 없는 2글자 단어를 정확히 읽는다.
- 단어를 보고 소리 내어 또박또박 쓴다.

## 핵심 학습
- 단어: 나무, 모자, 바지, 사과, 우유, 하마

## 15분 수업 구성
### 1단계 (5분) 자음/모음 읽기
- 오늘 단어에 나오는 핵심 음절 읽기(`나/무`, `모/자` 등)
- 빠른 카드 읽기 10회

### 2단계 (5분) 단어 읽기
- 그림 보고 단어 읽기
- 단어-그림 연결 퀴즈 6문제

### 3단계 (5분) 따라 쓰기
- 단어 3개 선택 따라 쓰기
- 완성 단어를 소리 내어 다시 읽기

## 웹 서비스 적용 가이드
- 좌측 패널: 그림+단어 쌍 카드
- 우측 캔버스: 단어 단위 트레이싱
- 성공 피드백: 단어별 성공 도장

## 확인 체크
- 6개 단어 중 4개 이상 정확히 읽는다.
- 선택 단어 3개를 바르게 쓴다.
