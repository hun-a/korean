# 012일차 - 모아쓰기 2 (`고/구/기` 패턴)

## 오늘 목표
- 같은 자음에 다른 모음을 붙여 읽는다.
- `고`, `구`, `기` 패턴 단어를 읽고 쓴다.

## 핵심 학습
- 자모: `ㄱ`, `ㄴ`, `ㅁ` + `ㅗ/ㅜ/ㅣ`
- 단어: 고구마, 구두, 기차, 누나, 미소

## 15분 수업 구성
### 1단계 (5분) 자음/모음 읽기
- `고-구-기`, `노-누-니`, `모-무-미` 읽기
- 모음만 바뀌는 점 강조

### 2단계 (5분) 단어 읽기
- 패턴 단어 5개 읽기
- 같은 시작 글자 단어끼리 묶기

### 3단계 (5분) 따라 쓰기
- `고 구 기` 계열 음절 6개 쓰기
- 단어 2개 따라 쓰기

## 웹 서비스 적용 가이드
- 좌측 패널: 같은 초성 묶음 카드
- 우측 캔버스: 모음 교체형 쓰기 연습 슬롯
- 성공 피드백: 3연속 정답 시 폭죽

## 확인 체크
- 모음 변화에 따라 소리가 달라짐을 안다.
- 패턴 음절을 읽고 쓴다.
