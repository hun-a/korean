# 030일차 - 30일 종합 발표와 마무리

## 오늘 목표
- 30일 학습 내용을 읽기/쓰기로 종합 확인한다.
- 자신 있게 발표하고 성취감을 느낀다.

## 핵심 학습
- 종합 단어: 나무, 학교, 친구, 하늘, 공원
- 종합 문장: `저는 한글을 읽고 쓸 수 있어요.`

## 15분 수업 구성
### 1단계 (5분) 자음/모음 읽기
- 랜덤 자모 10개 읽기
- 약한 자모 2개 즉시 복습

### 2단계 (5분) 단어 읽기
- 종합 단어 5개 읽기
- 자신 있는 단어 1개 발표

### 3단계 (5분) 따라 쓰기
- 종합 문장 1개 따라 쓰기
- 마지막에 자기 이름 서명하기

## 웹 서비스 적용 가이드
- 좌측 패널: 30일 누적 성취 대시보드
- 우측 캔버스: 종합 문장 쓰기 + 자동 채점
- 성공 피드백: 대형 폭죽, 박수, 수료 배지

## 확인 체크
- 기본 자모/단어를 안정적으로 읽는다.
- 문장 쓰기와 발표를 완주한다.
