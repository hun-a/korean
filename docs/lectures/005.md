# 005일차 - 복합모음 `ㅐ`, `ㅔ`

## 오늘 목표
- `ㅐ`, `ㅔ`를 읽고 쓸 수 있다.
- `애`, `에`가 들어간 쉬운 단어를 읽는다.

## 핵심 학습
- 자모: `ㅐ`, `ㅔ`
- 단어: 새, 세수, 애기, 에어컨

## 15분 수업 구성
### 1단계 (5분) 자음/모음 읽기
- `ㅐ`와 `ㅔ` 카드 번갈아 읽기
- 모음 복습 카드와 섞어 빠르게 인식하기

### 2단계 (5분) 단어 읽기
- `새`, `세수`, `애기`, `에어컨` 읽기
- 같은 모음 들어간 단어끼리 묶기 놀이

### 3단계 (5분) 따라 쓰기
- `ㅐ`, `ㅔ`, `애`, `에`, `새`, `세` 따라 쓰기
- 정확한 획순 시 박수, 전부 완료 시 별비 애니메이션

## 웹 서비스 적용 가이드
- 좌측 패널: 단어 아래 모음 색상 표시
- 우측 캔버스: 글자 겹치기(트레이싱) 모드
- 성공 피드백: 모음 구분 성공률 퍼센트 표시

## 확인 체크
- `ㅐ/ㅔ`를 보고 구분한다.
- `새/세`를 읽고 쓴다.
