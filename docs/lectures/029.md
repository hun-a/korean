# 029일차 - 나만의 한글책 만들기

## 오늘 목표
- 배운 단어와 문장을 모아 작은 책을 만든다.
- 읽기와 쓰기 성취를 시각적으로 확인한다.

## 핵심 학습
- 구성: 표지 1장 + 단어 2장 + 문장 1장

## 15분 수업 구성
### 1단계 (5분) 자음/모음 읽기
- 내가 고른 단어의 핵심 글자 읽기
- 어려운 글자 3개 복습

### 2단계 (5분) 단어 읽기
- 책에 넣을 단어 3개 읽기
- 단어 순서 정하기(쉬운 -> 어려운)

### 3단계 (5분) 따라 쓰기
- 단어 2개 + 문장 1개 따라 쓰기
- 완성 페이지에 스티커 붙이기

## 웹 서비스 적용 가이드
- 좌측 패널: 책 페이지 템플릿(표지/단어/문장)
- 우측 캔버스: 페이지별 쓰기 저장
- 성공 피드백: 페이지 완료마다 폭죽, 완성 시 트로피

## 확인 체크
- 스스로 고른 단어를 읽는다.
- 책 페이지 3장 이상 완성한다.
