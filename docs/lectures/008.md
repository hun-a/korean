# 008일차 - 자음 `ㄱ`, `ㄴ`, `ㄷ`, `ㄹ`

## 오늘 목표
- 기본 자음 4개를 읽고 쓸 수 있다.
- `가/나/다/라`를 읽는다.

## 핵심 학습
- 자모: `ㄱ`, `ㄴ`, `ㄷ`, `ㄹ`
- 단어: 가방, 나비, 다리, 라면

## 15분 수업 구성
### 1단계 (5분) 자음/모음 읽기
- 자음 이름 읽기 + `ㅏ`와 결합해 `가/나/다/라` 읽기
- 자음 카드 순서 바꾸기 퀴즈

### 2단계 (5분) 단어 읽기
- `가방`, `나비`, `다리`, `라면` 그림 단어 읽기
- 단어 첫 글자 맞히기 클릭 게임

### 3단계 (5분) 따라 쓰기
- `ㄱ ㄴ ㄷ ㄹ`과 `가 나 다 라` 따라 쓰기
- 8개 완료 시 박수와 별 1개 지급

## 웹 서비스 적용 가이드
- 좌측 패널: 초성 강조(첫 글자 확대)
- 우측 캔버스: 자음 전용 획순 트레이싱
- 성공 피드백: 첫 글자 정답 연속 시 폭죽

## 확인 체크
- 자음 4개를 모양으로 구분한다.
- `가/나/다/라`를 읽고 쓴다.
