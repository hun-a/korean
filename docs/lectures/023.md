# 023일차 - 받아쓰기 1 (단어 중심)

## 오늘 목표
- 들은 단어를 떠올려 쓴다.
- 읽기와 쓰기를 연결한다.

## 핵심 학습
- 단어: 바람, 하늘, 나무, 고양이, 학교, 친구

## 15분 수업 구성
### 1단계 (5분) 자음/모음 읽기
- 단어 핵심 글자(하, 학, 친, 구 등) 읽기
- 비슷한 글자 구분 퀴즈

### 2단계 (5분) 단어 읽기
- 전체 단어 먼저 눈으로 읽고 의미 확인
- 발음 듣기 버튼으로 1회 복습

### 3단계 (5분) 따라 쓰기
- 받아쓰기 모드: 소리 듣고 쓰기 3문항
- 정답 비교 후 오답 1회 재쓰기

## 웹 서비스 적용 가이드
- 좌측 패널: 받아쓰기 오디오 플레이어
- 우측 캔버스: 정답 오버레이 ON/OFF
- 성공 피드백: 만점 시 폭죽 + 메달 배지

## 확인 체크
- 소리 듣고 단어를 2개 이상 정확히 쓴다.
- 오답을 보고 바로 수정한다.
