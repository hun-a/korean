# 021일차 - 쌍자음 단어 읽기

## 오늘 목표
- 쌍자음이 들어간 단어를 자연스럽게 읽는다.
- 단어를 따라 쓰며 정확도를 높인다.

## 핵심 학습
- 단어: 꼬리, 따뜻, 빵, 씨앗, 짜장

## 15분 수업 구성
### 1단계 (5분) 자음/모음 읽기
- 오늘 단어의 쌍자음 글자만 모아 읽기
- 쌍자음 위치(첫소리/가운데) 확인

### 2단계 (5분) 단어 읽기
- 단어 5개 읽기
- 교사가 말한 단어를 화면에서 고르는 게임

### 3단계 (5분) 따라 쓰기
- 단어 3개 선택 따라 쓰기
- 맞으면 박수, 틀리면 획순 애니메이션 재생

## 웹 서비스 적용 가이드
- 좌측 패널: 쌍자음 하이라이트 강조
- 우측 캔버스: 단어 단위 자동 채점
- 성공 피드백: 연속 정답 3회 시 별비

## 확인 체크
- 쌍자음 단어를 천천히라도 정확히 읽는다.
- 단어 3개를 바르게 쓴다.
