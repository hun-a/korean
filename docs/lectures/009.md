# 009일차 - 자음 `ㅁ`, `ㅂ`, `ㅅ`, `ㅇ`

## 오늘 목표
- `ㅁ`, `ㅂ`, `ㅅ`, `ㅇ`을 읽고 쓸 수 있다.
- `마/바/사/아`를 단어에 적용한다.

## 핵심 학습
- 자모: `ㅁ`, `ㅂ`, `ㅅ`, `ㅇ`
- 단어: 모자, 바다, 사자, 아이

## 15분 수업 구성
### 1단계 (5분) 자음/모음 읽기
- `ㅁ ㅂ ㅅ ㅇ` 읽고 `ㅏ` 결합 음절 읽기
- `ㅁ/ㅂ` 구분 미니 퀴즈

### 2단계 (5분) 단어 읽기
- `모자`, `바다`, `사자`, `아이` 읽기
- 단어 속 첫소리 찾기 놀이

### 3단계 (5분) 따라 쓰기
- `ㅁ ㅂ ㅅ ㅇ`과 `마 바 사 아` 따라 쓰기
- 정확도 80% 이상이면 박수 애니메이션

## 웹 서비스 적용 가이드
- 좌측 패널: 첫소리 하이라이트
- 우측 캔버스: 비교 쓰기(`ㅁ` vs `ㅂ`) 영역
- 성공 피드백: 비교 문제 정답 시 배지 지급

## 확인 체크
- `ㅁ/ㅂ`를 헷갈리지 않는다.
- 4개 단어를 읽고 핵심 글자를 쓴다.
