# 028일차 - 나를 소개하는 문장

## 오늘 목표
- 자기 이름과 좋아하는 것을 읽고 쓴다.
- 1~2문장 자기소개를 완성한다.

## 핵심 학습
- 문장 틀: `제 이름은 ___예요.`, `저는 ___를 좋아해요.`

## 15분 수업 구성
### 1단계 (5분) 자음/모음 읽기
- 자기 이름 글자 분해 읽기
- 소개 문장 핵심 음절 읽기

### 2단계 (5분) 단어 읽기
- 좋아하는 것 단어 카드 읽기(공, 책, 고양이 등)
- 문장 틀에 단어 넣어 소리 내어 읽기

### 3단계 (5분) 따라 쓰기
- 자기소개 문장 1~2개 따라 쓰기
- 이름 글자는 크게 한 번 더 쓰기

## 웹 서비스 적용 가이드
- 좌측 패널: 이름 입력 후 자동 글자 분해
- 우측 캔버스: 문장 틀 자동 채움 + 손글씨 영역
- 성공 피드백: 자기소개 완성 카드 + 박수

## 확인 체크
- 자기 이름을 정확히 읽는다.
- 자기소개 문장 1개 이상 쓴다.
