# 015일차 - 1~14일차 중간 점검

## 오늘 목표
- 모음, 자음, 기본 단어 읽기를 점검한다.
- 약한 글자를 찾아 다시 쓰기 연습한다.

## 핵심 학습
- 모음 전 범위 + 기본 자음
- 단어: 나무, 바지, 우유, 고구마, 하늘, 포도

## 15분 수업 구성
### 1단계 (5분) 자음/모음 읽기
- 랜덤 자모 15개 읽기
- 틀린 자모는 즉시 다시 읽기

### 2단계 (5분) 단어 읽기
- 복습 단어 6개 읽기 테스트
- 정확도 낮은 단어 2개 재도전

### 3단계 (5분) 따라 쓰기
- 틀린 자모 2개 + 단어 2개 따라 쓰기
- 마지막 30초 자기이름 쓰기

## 웹 서비스 적용 가이드
- 좌측 패널: 정확도 대시보드(읽기 점수)
- 우측 캔버스: 개인 취약 글자 자동 추천
- 성공 피드백: 전주 대비 향상 시 특별 폭죽

## 확인 체크
- 읽기 정확도 70% 이상 달성
- 취약 글자를 스스로 고쳐 쓴다.
