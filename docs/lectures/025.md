# 025일차 - 그림 보고 문장 만들기

## 오늘 목표
- 그림을 보고 알맞은 단어/문장을 읽는다.
- 한 문장을 따라 쓰며 표현을 확장한다.

## 핵심 학습
- 그림 주제: 놀이터, 학교, 집
- 문장 예시: `아이가 뛰어요.`, `친구가 웃어요.`

## 15분 수업 구성
### 1단계 (5분) 자음/모음 읽기
- 오늘 문장 핵심 음절(`아`, `이`, `뛰`, `웃`) 읽기
- 어려운 음절 3개 반복

### 2단계 (5분) 단어 읽기
- 그림 속 단어 읽기(아이, 친구, 공, 학교)
- 단어를 조합해 문장 소리 내어 읽기

### 3단계 (5분) 따라 쓰기
- 예시 문장 1개 따라 쓰기
- 마지막 1분: 단어 1개 바꿔 다시 쓰기

## 웹 서비스 적용 가이드
- 좌측 패널: 그림 + 추천 단어 버튼
- 우측 캔버스: 문장 자동 줄맞춤
- 성공 피드백: 문장 완성 시 캐릭터 춤 애니메이션

## 확인 체크
- 그림과 단어를 연결해 읽는다.
- 문장 1개를 정확히 쓴다.
