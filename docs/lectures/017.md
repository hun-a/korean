# 017일차 - 받침 기초 2 (`ㅁ`, `ㅂ`, `ㅇ`)

## 오늘 목표
- `ㅁ`, `ㅂ`, `ㅇ` 받침 단어를 읽는다.
- 받침 있는 단어를 또박또박 쓴다.

## 핵심 학습
- 받침: `ㅁ`, `ㅂ`, `ㅇ`
- 단어: 밤, 집, 공, 곰, 밥

## 15분 수업 구성
### 1단계 (5분) 자음/모음 읽기
- `ㅁ`, `ㅂ`, `ㅇ` 자음 읽기
- `마-맘`, `바-밥`, `고-공` 비교 읽기

### 2단계 (5분) 단어 읽기
- `밤`, `집`, `공`, `곰`, `밥` 읽기
- 그림 보고 받침 단어 고르기

### 3단계 (5분) 따라 쓰기
- `밤 집 공 밥` 따라 쓰기
- 오답 글자는 즉시 재트레이싱

## 웹 서비스 적용 가이드
- 좌측 패널: 받침별 색상 태그
- 우측 캔버스: 받침 쓰기 확대 모드
- 성공 피드백: 5단어 완료 시 별비 애니메이션

## 확인 체크
- `ㅁ/ㅂ/ㅇ` 받침 소리를 구분한다.
- 4개 단어를 바르게 쓴다.
