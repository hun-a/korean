# 014일차 - 받침 없는 단어 읽기 2

## 오늘 목표
- 새로운 2~3글자 단어를 읽는다.
- 쉬운 문장(2어절)로 확장한다.

## 핵심 학습
- 단어: 다리미, 고구마, 나비, 바나나, 머리
- 문장: `나비가 와요.`, `고구마가 커요.`

## 15분 수업 구성
### 1단계 (5분) 자음/모음 읽기
- 단어 구성 음절 분리 읽기(`고-구-마` 등)
- 리듬 읽기로 속도 높이기

### 2단계 (5분) 단어 읽기
- 단어 5개 읽기 후 2어절 문장 2개 읽기
- 문장 속 단어 찾기 게임

### 3단계 (5분) 따라 쓰기
- 단어 2개 + 문장 1개 따라 쓰기
- 문장 끝 마침표까지 쓰기

## 웹 서비스 적용 가이드
- 좌측 패널: 음절 분절 버튼 제공
- 우측 캔버스: 문장 칸(띄어쓰기 힌트 점선)
- 성공 피드백: 문장 완성 시 폭죽 + 박수

## 확인 체크
- 3글자 단어를 끊어 읽을 수 있다.
- 짧은 문장 1개를 정확히 쓴다.
