# 019일차 - 받침 포함 짧은 문장 읽기

## 오늘 목표
- 받침 단어가 들어간 2~3어절 문장을 읽는다.
- 짧은 문장을 따라 쓴다.

## 핵심 학습
- 문장: `달이 커요.`, `공이 굴러요.`, `밥을 먹어요.`

## 15분 수업 구성
### 1단계 (5분) 자음/모음 읽기
- 문장 핵심 글자(`달`, `공`, `밥`) 분해 읽기
- 받침 글자만 모아 빠르게 읽기

### 2단계 (5분) 단어 읽기
- 문장 속 단어를 먼저 읽고 문장으로 확장
- 단어 카드 순서 맞추기 놀이

### 3단계 (5분) 따라 쓰기
- 문장 1개 선택해 따라 쓰기
- 띄어쓰기 칸을 보고 정확히 쓰기

## 웹 서비스 적용 가이드
- 좌측 패널: 문장 -> 단어 -> 글자 단계 전환 버튼
- 우측 캔버스: 띄어쓰기 점선 가이드
- 성공 피드백: 문장 완성 시 박수 + 칭찬 음성

## 확인 체크
- 받침 문장을 끊어 읽는다.
- 문장 끝 마침표까지 쓴다.
